# To-Do List App (Python)

## Description

This is a simple command-line To-Do List application built using Python.
It helps users manage daily tasks by allowing them to add, view, and delete tasks.
---
## Features

* Add new tasks
* View all tasks
* Delete tasks
* Saves tasks in a file (tasks.txt)
---
## Technologies Used

* Python
* File Handling
* Basic CLI (Command Line Interface)
---
## Project Structure

todo.py       # Main program
tasks.txt     # Stores tasks
